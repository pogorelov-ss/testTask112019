============
Installation
============


Stable release
--------------

To install testTask112019, run this command in your terminal:

.. code-block:: console

    $ pip install testtask112019

This is the preferred method to install testTask112019, as it will always install the most recent stable release.

If you don't have `pip`_ installed, this `Python installation guide`_ can guide
you through the process.

.. _pip: https://pip.pypa.io
.. _Python installation guide: http://docs.python-guide.org/en/latest/starting/installation/


From sources
------------

The sources for testTask112019 can be downloaded from the `Github repo`_.

You can either clone the public repository:

.. code-block:: console

    $ git clone git://github.com/pogorelov-ss/testtask112019

Or download the `tarball`_:

.. code-block:: console

    $ curl  -OL https://github.com/pogorelov-ss/testtask112019/tarball/master

Once you have a copy of the source, you can install it with:

.. code-block:: console

    $ python setup.py install

.. _tarball: https://github.com/pogorelov-ss/testtask112019/tarball/master
